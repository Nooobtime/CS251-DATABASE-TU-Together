# Automate test Protractor

use npm to install Protractor globally with

```sh
npm install -g protractor
```

update webdriver

```sh
webdriver-manager update
```

Now start up a server with

```sh
webdriver-manager start
```

run the test

```sh
protractor conf.js
```
